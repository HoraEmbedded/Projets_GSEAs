Projets Académiques & Techniques

■ Programmation DSP — TMS320C6713 DSK

Mise en œuvre du DSP TI TMS320C6713 sous Code Composer Studio v7. Développement en C et assembleur C6000 pour la génération de signaux et le filtrage numérique FIR/IIR en temps réel. Diagnostic de dépendances de compilation et maîtrise du workflow toolchain DSP embarqué.

Consulter le code
C / C++ Assembleur C6000 TMS320C6713 Traitement du Signal

■ Étude et Dimensionnement d'un Transistor NMOS en Commutation

Analyse, calcul théorique et dimensionnement d'un canal de commande de puissance utilisant un transistor NMOS en régime de commutation. Choix et validation des composants associés pour assurer la robustesse du montage : sélection d'un optocoupleur pour l'isolation galvanique, d'une diode rapide pour la protection contre les surtensions inductives, et calcul des puissances dissipées par les résistances de polarisation.

Consulter le code
Électronique de Puissance Dimensionnement HW Composants CMS Bureau d'Études

■ Smart Greenhouse Management System (Bare-Metal)

Conception d'un système de gestion de serre automatisée sur microcontrôleur ATmega2560. Développement en C pur (Bare-Metal) avec architecture temps-réel pilotée par interruptions matérielles (Timer1). Implémentation d'un contrôle par hystérésis à double seuil et d'arithmétique à virgule fixe pour optimiser les performances et la durée de vie des relais. Simulation "Software-in-the-Loop" (VS Code, Wokwi) avec monitoring I2C/UART.

Consulter le code
C (Bare-Metal) ATmega2560 Wokwi / PlatformIO Systèmes Embarqués

■ Modélisation et Commande de Robot Manipulateur (2DDL)

Modélisation physique et implémentation d'une commande articulaire PID pour un robot à 2 degrés de liberté sous MATLAB/Simscape Multibody. Étude de la cinématique directe (FK) et inverse (IK) pour la génération de trajectoires. Optimisation de la robustesse et du suivi de trajectoire, atteignant une précision avec une erreur latérale inférieure à 0.05 m.

Consulter le code
MATLAB / Simscape PID Cinématique (FK/IK) Robotics System Toolbox Robotique Industrielle

■ Les Auto-Encodeurs (AE) — Compression & Détection d'Anomalies

Étude approfondie et implémentation d'architectures d'apprentissage profond (Vanilla AE, Sparse, Denoising, et VAE). Application pratique dédiée au traitement d'images (filtrage du bruit) et au diagnostic industriel par détection d'anomalies, complétée par une analyse comparative des performances des modèles.

Consulter le code
Deep Learning Auto-encodeurs (VAE) Python / Keras Machine Learning

■ Convoyeur de tri automatique (Factory I/O)

Programmation d'un automate Siemens S7-1200 (simulé) via TIA Portal pour trier des objets par couleur. Développement du programme Ladder avec gestion des modes automatique, manuel et arrêt d'urgence. Communication PLCSIM ↔ Factory I/O via OPC-DA.
