📁 Projets Académiques & Techniques

🤖 Contrôle Flou Mamdani — Bras Robotique 2-DOF

Conception et implémentation d'un contrôleur Flou PD de type Mamdani pour la commande d'un bras robotique à 2 degrés de liberté sous MATLAB/Simulink. Optimisation des fonctions d'appartenance et des règles floues pour minimiser l'erreur de position angulaire. Comparaison des performances avec un PID classique.

📁 Consulter le rapport / code
MATLAB Simulink Fuzzy Logic Toolbox Contrôle Intelligent Robotique

🧠 Approximation par Réseau de Neurones (MLP)

Implémentation d'un contrôleur à base de réseau de neurones artificiels (MLP – Perceptron Multicouche) pour approximer le comportement optimal d'un contrôleur sur bras robotique 2-DOF. Entraînement supervisé avec données générées par contrôleur de référence. Intégration dans Simulink via un bloc personnalisé.

📁 Consulter le rapport / code
MATLAB Neural Network Toolbox Simulink Machine Learning Contrôle

🧬 Optimisation PID par Algorithme Génétique (GA)

Optimisation automatique des paramètres (Kp, Ki, Kd) d'un contrôleur PID via Algorithme Génétique pour la commande d'un bras robotique 2-DOF. Définition de la fonction de fitness basée sur l'ITAE. Comparaison des performances avec tuning manuel et méthodes analytiques.

📁 Consulter le rapport / code
MATLAB Global Optimization Toolbox Optimisation Algorithme Génétique

🔧 Implémentation S-Function MATLAB/Simulink

Développement d'une S-Function C personnalisée sous MATLAB/Simulink pour intégrer des algorithmes de contrôle avancés dans l'environnement de simulation temps continu. Application sur le modèle de bras robotique 2-DOF. Gestion des entrées/sorties, états discrets et continus.

📁 Consulter le rapport / code
MATLAB C/C++ S-Function Simulation Temps-réel

💾 Contrôleur PID sur FPGA

Conception et implémentation d'un contrôleur PID numérique sur FPGA en VHDL. Synthèse, simulation fonctionnelle et temporelle avec Quartus II. Validation hardware sur carte de développement. Optimisation de la précision numérique (virgule fixe).

💾 Code VHDL
VHDL Quartus II FPGA Embarqué Numérique

⚡ Convertisseur Buck — Conception PCB

Conception complète d'un convertisseur abaisseur de tension (Buck) incluant le dimensionnement des composants, simulation sous LTSpice et réalisation du PCB sur plusieurs couches. Validation des performances (rendement, ondulation, réponse transitoire) par mesures expérimentales sur banc de test.

📐 Voir les schémas
LTSpice Altium / KiCad Électronique de Puissance Conception PCB

🌐 Projet IoT Embarqué

Développement d'un système IoT complet incluant acquisition de données via capteurs (température, humidité, luminosité), traitement local sur microcontrôleur et transmission sans fil. Interface de monitoring avec affichage temps réel et alertes configurables. Communication MQTT sur réseau Wi-Fi.

📁 Consulter le code
ESP32 Arduino IDE / C++ MQTT IoT Temps Réel
